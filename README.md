## どんなゲーム？

錬金術で生まれた100体の幻獣を集め、育て、進化させる育成RPGです。
9属性の相性を突く戦い、才能値による厳選、1/100で出会える変異種（色違い）、
そしてエンドコンテンツ「試練の塔」。ポケモン・DQM世代に懐かしくも新しい、
スキマ時間で遊べる一作です。

- 🧪 **全100体の幻獣** — 捕獲・進化・錬成で図鑑を埋める
- ⚔️ **9属性の相性バトル** — 火水風地雷毒聖冥錬成。弱点を突く駆け引き
- ✨ **才能値と変異種** — ★厳選＋1/100の色違いレア
- 🗼 **試練の塔** — 何階まで行ける？ スコアをSNSでシェア
- 💾 **セーブはブラウザ内** — サーバー不要・アカウント不要

## 遊び方

上のリンクを開くだけ。移動はスワイプ／矢印キー、あとは画面の指示に従って冒険を進めてください。
セーブはお使いのブラウザに自動保存されます（別端末には引き継がれません）。

## 技術構成

- React 18 + TypeScript + Vite 5
- バックエンドなし（状態は localStorage に保存）
- GitHub Pages へ自動デプロイ（main への push で公開）
- 戦闘エンジンは純粋関数化し、ヘッドレスでバランスシミュレーション可能

## 開発

```bash
npm install
npm run dev      # 開発サーバ (http://localhost:5173)
npm run build    # 型チェック + RNGリント + 本番ビルド
npm run preview  # 本番ビルドのプレビュー
```
