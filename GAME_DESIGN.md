# 錬金幻獣録 アルケミスト・オーダー — ゲームデザイン仕様書 (GDD)

> モンスター育成RPG / 舞台: 錬金術が栄えた架空の近世西洋「アルケミア大陸」
> 幻獣総数: 約100体 / 属性9種 / 最大3段階進化

---

## 1. 世界観

錬金術師たちは大気に満ちる元素粒子「**エーテル**」から、幻のごとき獣 ——
**幻獣 (Phantasma)** を錬成する術を確立した。彼ら幻獣使いは
**錬獣師 (Alchemus)** と呼ばれ、その頂点に立つ秩序が **アルケミスト・オーダー**。

幻獣はフラスコや使い魔の核から生まれ、育成と「**転成 (Transmutation)**」=進化を
通じて姿を変える。錬獣師は契約した幻獣と旅をし、野生の幻獣を捕え、
腕を競い合う。

### 中心概念: 賢者の石

すべての幻獣の源は伝説の触媒「**賢者の石 (Lapis)**」。
完全な賢者の石は「生命を無限に錬成する」とされ、これを巡って物語が動く。

## 2. ストーリー (メインプロット)

### 発端
主人公は地方の錬金工房の見習い。師から最初の幻獣（御三家）の核を授かり、
オーダーの登録試験を受けるため旅立つ。

### 中盤の謎: 「灰化現象 (The Greying)」
各地で幻獣が色を失い、暴走して「**灰の幻獣 (Cinereal)**」と化す事件が多発。
原因は、かつてオーダーを追放された天才錬獣師 **マグヌス** が起こした
**大錬成事故** ——不完全な賢者の石が暴走し、世界のエーテルを汚染している。

### クライマックス
マグヌスは「全幻獣を一体の完全生命に統合する」狂気の理想を掲げ、
セレスティアル・キメラを核とした禁忌の存在 **アビス・キメラ** を錬成。
主人公は育てた幻獣たちと共に、汚染の中心「**大錬成炉**」へ挑む。

### ラスボス
- **第1形態: 灰王 マグヌス** (錬成タイプ)
- **最終形態: 禁忌の幻獣 アビス・キメラ** (冥/聖の複合・特殊個体)

### 結末分岐 (実装で拡張可能)
賢者の石を「破壊する」か「浄化して継承する」かでエンディング分岐。

### 相性チャート (防御側が受けるダメージ倍率) ※要バランス調整

基本サイクル（元素4すくみ）: **水→火→風→地→水** が2倍。

| 属性 | 弱点(2.0x) | 半減(0.5x) |
|------|-----------|-----------|
| 火 | 水 / 地 | 火 / 風 / 錬成 |
| 水 | 雷 / 地 | 水 / 火 |
| 風 | 雷 / 火 | 風 / 地 |
| 地 | 水 / 風 | 雷 / 毒 |
| 雷 | 地 | 雷 / 風 |
| 毒 | 火 / 聖 | 毒 / 地 |
| 聖 | 冥 | 聖 / 火 |
| 冥 | 聖 / 火 | 冥 / 毒 |
| 錬成 | 火 / 雷 | 錬成 / 聖 |

> 設計意図: 序盤は元素4すくみで分かりやすく、聖⇔冥と錬成で終盤の戦略性を上げる。

## 4. 育成・バトルシステム

### ステータス
HP / こうげき(ATK) / ぼうぎょ(DEF) / すばやさ(SPD) / かしこさ(MAG・特殊攻撃)

### 進化（転成）
- レベル到達 / 特定アイテム「**転成触媒**」/ なつき度 などをトリガー。
- 進化系統は最大3段階。一部は1〜2段階や、条件分岐進化（例: キメラ系）。

### 技 (錬成術)
各幻獣は属性技を4つまで習得。技には威力・命中・PP・属性。

### 捕獲
野生戦で弱らせ「**封獣フラスコ**」を投げる（HPと状態異常で成功率変動）。

### 状態異常
やけど / こおり / まひ / どく / 眠り / 灰化（このゲーム独自・徐々にステ低下）。

## 5. ゲーム進行 (マップ構成案)

1. **始まりの村ラピス** … 御三家入手・チュートリアル
2. **緑霧の森** … 地/毒の幻獣 / 最初の灰化事件
3. **潮鳴りの港町** … 水系 / ジム戦その1
4. **嵐嶺グランドピーク** … 風系 / グランドロク
5. **錆びた工廠都市** … 錬成系 / マグヌスの痕跡
6. **黄昏の墓所** … 冥系 / ヴァンプ・ロード
7. **天文台アストラ** … 聖系 / キメラの伝承
8. **大錬成炉（最終ダンジョン）** … ラスボス マグヌス & アビス・キメラ

各町に「オーダー守護者（=ジムリーダー的存在）」を配置し、勝つと記章を獲得。
記章8個でオーダー本部へ。
