# 錬金幻獣録 アルケミスト・オーダー — システム定義（数値・計算式）

実装: `src/game/state.ts`(経験値/成長/捕獲)・`src/engine/battleEngine.ts`(ダメージ/状態異常)・`src/game/moves.ts`(技/習得)・`data/typechart.json`(相性)。

**具体的な一覧表(出現プール/特性/もちもの/錬成レシピ)は `docs/DATA_TABLES.md` を参照**（実データから自動生成・`npx tsx scripts/gen_data_docs.mjs`で再生成）。本書は計算式・ルールの説明に専念する。

## 1. レベルと経験値

### 経験値曲線
- **次のレベルまでの必要経験値**: `expToNext(L) = 12 + L × 8`
- **レベルLに到達する累計経験値**: `(L−1) × (4L + 12)`（2次曲線。後半ほど重く"出来事感"）
- **撃破で得る経験値**: `expReward(敵Lv) = 18 + 敵Lv × 6`

| Lv | 次まで | 累計 | 例: 同Lv敵を倒した時の取得 |
|----|-------|------|--------------------------|
| 5  | 52    | 64   | 48 |
| 10 | 92    | 468  | 78 |
| 20 | 172   | 1748 | 138 |
| 30 | 252   | 3768 | 198 |
| 50 | 412   | 10388| 318 |

- 上限レベル **100**。御三家開始 **Lv8**。

### ステータス成長（種族値→実数）
種族値 `[HP,ATK,DEF,SPD,MAG]`（`data/monsters.json`）から:
- **HP** = `floor(2×種族値×Lv/100) + Lv + 10`
- **他** = `floor(2×種族値×Lv/100) + 5`

例（ヴォルカドン 種族値[95,118,92,100,95]）:
| Lv | HP | ATK | DEF | SPD | MAG |
|----|----|----|----|----|----|
| 16 | 56 | 42 | 34 | 37 | 35 |
| 30 | 97 | 75 | 60 | 65 | 62 |
| 50 |150 |123 | 97 |105 |100 |

## 2. 技と習得（レベルで覚える）

- **最大4枠**。レベルが上がると新技を覚え、**古い技と入れ替わる**（新しく覚えた順に4つ保持）。
- 種ごとの習得表 `learnset(species)`（`moves.ts`。**2026-07パッケージAで5枠制に改訂**）:

| 習得Lv | 技 |
|-------|----|
| 1 | たいあたり（主属性・物理40） |
| 1 | 主属性の基本技(basic・物理/特殊 45〜50) |
| 7 | 主属性の特殊技(tech・状態異常 or 回復) |
| 10 | 主属性の連撃/先制/反動技(rush) |
| 14 | 主属性のランク操作/ガード技(aura) |
| 16 | （複合タイプのみ）副属性の強技(blast) |
| 18 | 主属性の強技(blast・威力80〜90) |
| 22 | **固有技**（特殊・威力85前後／その幻獣の代名詞。雷タイプは会心率UP付き） |

- レベルアップ時、新技を覚えると「技【〇〇】を おぼえた！」と通知（`grantExp`）。
- 技は **威力・命中・カテゴリ（物理/特殊/状態）・属性** に加え、**技タグ**(priority/multi/recoil/drain/guard/charge/critBoost/bonusVsStatus/buffs/resetStages)を持てる。詳細は`docs/SPEC_BATTLE_DEPTH.md`。

### タイプ別の技キット（`TYPE_KIT`・5枠: basic/blast/tech/rush/aura）
各タイプの技タグ個性(抜粋。全表は`docs/SPEC_BATTLE_DEPTH.md`§5):
火=高火力+反動(rush) / 水=デバフ+防御↑(aura) / 風=先制+速さ↑(aura) / 地=連撃+ガード(aura) / 雷=高会心+まりょく↑(aura) / 毒=状態異常特効+デバフ(aura) / 聖=回復+バフ2種(aura) / 冥=吸収+デバフ(aura) / 錬成=溜め高火力+能力リセット(aura)。
状態異常の割当は変わらず: 火=やけど / 水=こおり / 雷=まひ / 毒=どく / 地=ねむり / 冥=灰化 / 聖=回復・状態治癒 / 錬成=自己修復。

## 3. ダメージ計算

```
atkStat = round(ATK or MAG × stageMult(能力ランク))  // やけど半減・剛力補正はこの後
defStat = max(1, round(DEF × stageMult(能力ランク)))
base = floor( (2×Lv/5 + 2) × 威力 × (atkStat/defStat) / 50 + 2 )
ダメージ = base × タイプ一致(1.5) × 相性倍率 × 乱数(0.85〜1.0) × 各種補正
```
- 攻撃ステ: 物理=ATK / 特殊=MAG。防御ステ=DEF。
- 補正: やけど中の物理は攻撃半減。灰化中は与ダメージ×0.85。相性0倍は無効(ダメージ0)。ガード中は被ダメ×0.3。
- **能力ランク**(パッケージA追加): ±3段階、1段=stageMult(±33%乗算)。交代/登場でリセット。
- **会心**(パッケージA追加): 基礎6%(`CRIT_BASE`)・技ごとに`critBoost`で上書き可・発生時ダメージ×1.5。相性0倍の技には発生しない。

### 相性（`typechart.json`・抜粋）
水→火/地、火→風/錬成/毒、地→火/雷/錬成、雷→水/風(地は無効0)、毒→聖、聖→冥/毒、冥→聖 …が2倍。同系統や金属系に半減/無効あり。

## 4. 状態異常

| 状態 | 効果 |
|------|------|
| やけど | 毎ターン最大HPの1/16減・物理攻撃半減 |
| どく | 毎ターン最大HPの1/16減 |
| 灰化（独自） | 毎ターン最大HPの1/12減・与ダメージ−15%（聖技で治癒可） |
| まひ | すばやさ半減・25%で行動不能 |
| こおり | 毎ターン25%で解凍、それまで行動不能 |
| ねむり | 2〜3ターン行動不能 |

- 既に状態異常中は重ねがけ不可。バトル終了で解除。

---

## 5. 捕獲

- 成功率 `catchChance = clamp(0.25 + (1 − 現HP/最大HP) × 0.6, 0.05, 0.95)`
- 伝説は ×0.35。HPを削るほど捕まえやすい。封獣フラスコを消費。

---

## 6. 持続HP・回復

- 各幻獣の現在HPは**バトル間で持続**（`OwnedMonster.hp`）。
- 回復手段: **宿屋**=全回復 / **傷薬**=最大HPの60%回復（フィールド/バトルの「どうぐ」）/ 敗北時は満タンで村へ。
