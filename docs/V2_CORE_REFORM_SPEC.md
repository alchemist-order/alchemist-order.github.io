# V2 Core Reform Spec — 操作レスで沼るキャラ収集ブラウザゲームへ

作成日: 2026-07-15  
目的: 「歩いて進むRPG」から、「魅力的な幻獣を眺め、育て、集め、編成し、報酬を受け取り続けるブラウザ育成ゲーム」へ重心を移す。

---

## 1. 目指すゲーム体験

### 一言で

**毎日3分で成果が出て、30分眺めても飽きない、幻獣コレクション育成ゲーム。**

### 現状の課題

- マップ移動・会話・階段・扉など、操作コストがブラウザゲームとしてやや重い。
- 幻獣300体、立ち絵、記章、メダル、素材などの資産が増えた一方で、ホーム画面上の「見せ場」がまだ弱い。
- 課金導線としてコイン販売を考えるなら、先に「コインを使いたくなる長期育成」と「無料でも楽しい信頼」が必要。

### 改革の方針

1. **ホームをゲームの中心にする**
   - マップは探索先の雰囲気演出へ格下げ。
   - プレイヤーが最も長く見る画面は、手持ち幻獣・報酬・派遣・錬成・実績が並ぶホームにする。

2. **移動操作を“選択式探索”へ置き換える**
   - 「森へ行く → 探索する → イベント3連 → バトル/捕獲/素材/宝箱」。
   - WASD/矢印移動は残してよいが、主導線ではなくする。

3. **キャラを常に前面に出す**
   - ホームの主役は選択中の幻獣。
   - 図鑑・詳細・戦闘結果・捕獲・錬成・派遣帰還すべてで大きく絵を出す。

4. **やりこみは“縦の深さ”を作る**
   - 図鑑300体
   - 才能値
   - 変異種
   - 記章8
   - 実績メダル
   - 素材錬成
   - 継承スロット
   - 週替わり塔
   - 派遣

5. **課金は最後に、慎重に**
   - 当面は販売しない。
   - コイン販売にはサーバー保存、決済、返金/表示、未成年配慮、資金決済法/景表法まわりの確認が必要。
   - 先に「無料で沼れる設計」を完成させる。

## 2. 新しいコアループ

### 旧ループ

マップ移動 → エンカウント → 戦闘 → 捕獲/報酬 → メニュー確認

### 新ループ

ホーム  
→ 探索先を選ぶ  
→ ワンタップ探索  
→ イベントカードが流れる  
→ 戦闘/捕獲/素材/宝箱  
→ 結果画面で幻獣を大きく見せる  
→ ホームに戻って育成/錬成/派遣/実績確認

### 1セッションの理想

- 10秒: ログイン報酬、派遣帰還、実績/記章の赤点を見る
- 1分: 探索を2〜3回
- 3分: 1体捕獲、素材獲得、日課達成
- 10分: 編成・錬成・図鑑確認
- 30分: 週替わり塔・厳選・素材周回

## 3. ホーム画面の再設計

### 最重要: “幻獣が主役”のホーム

ホーム上部に大きな「相棒カード」を置く。

表示要素:

- 幻獣の大きな画像
- 名前、属性、Lv、才能、変異種マーク
- ひとこと性格/フレーバー
- 次の成長まで
- 今日できること

例:

```text
┌────────────────────────┐
│  相棒: ルミリーフ Lv.18  ★★  │
│        [大きい幻獣画像]       │
│  「森の朝露が好き。今日は機嫌がいい」 │
│  EXP: █████░░  あと2戦       │
└────────────────────────┘
```

### ホームの主要ボタン

- 探索する
- 編成
- 図鑑
- 錬成
- 派遣
- 塔
- 記章/実績
- ショップ/支援（将来）

### 重要な考え方

マップを開かなくても、ゲームの9割が遊べる状態にする。  
マップは「世界観を浴びたい人向けのビュー」として残す。

## 4. 探索システム

### クイック探索

エリア選択:

- 緑霧の森
- 潮鳴りの海
- 紅蓮の火山郷
- 今後の世界

探索開始後、3〜5枚のイベントカードを表示:

- 幻獣の気配
- 素材発見
- 宝箱
- NPC会話
- ヌシの痕跡
- バトル

### 探索テンポ

- 1探索は30〜60秒
- スタミナなし
- ただし日課・週課・希少素材・レア枠で継続導線を作る

### 既存マップの扱い

- 既存の `Field.tsx` は残す。
- 新規導線として `ExploreScreen` / `ExploreModal` を追加。
- A/B確認後、マップ移動の主導線を弱める。

## 5. やりこみ設計

### 図鑑

300体の価値を最大化する。

- 未発見シルエット
- 捕獲済み
- 変異種
- 高才能
- 最終進化
- 入手場所
- お気に入り登録

### 個体厳選

既存の talent / mutant を主軸にする。

- 才能 0〜10
- 変異種 1/100
- お気に入りロック
- 高才能獲得演出
- 捕獲結果のシェアカード

### 錬成

素材24種・器4種・プレミアム素材を使う。

- 素材を使って才能/継承/進化ルートを強化
- 器で成功率や継承枠を変える
- 将来の課金コインは、ここに直接差し込まず、無料入手可能な補助アイテム購入に留める

### 派遣

低操作ゲームの主力。

- 手持ちから1〜3体を派遣
- 数分〜数時間後に帰還
- 素材、コイン、低確率で幻獣の痕跡
- 派遣中の幻獣にも愛着演出を付ける

### 週替わり塔

やりこみ層向け。

- 週ごとに属性/制限/報酬が変わる
- 自動生成に近い構造で長寿命化
- 報酬は記章風称号、メダル、素材、限定変異色など

### 画面ごとの主役

| 画面 | 主役 |
|---|---|
| ホーム | 相棒幻獣 |
| 探索 | 探索に出した幻獣 |
| 戦闘 | 大きな敵/味方スプライト |

| 派遣 | 帰ってきた幻獣 |

### 追加したい演出

- 幻獣の短い台詞/鳴き声風テキスト
- 好きな場所/好きな素材
- 派遣帰還コメント
- レア個体の特別演出
- 図鑑カードの背景テーマ

## 7. コイン販売の方針

### まず無料通貨として整える

現状の `money` を「ゲル」または「コイン」として整理する。  
販売予定の有料通貨とは混ぜないほうが安全。

推奨:

- `money`: 無料ゲーム内通貨
- `paidCoins`: 将来の有料通貨

### 販売前に必須

- サーバー保存
- アカウント
- 決済
- 購入履歴
- 返金対応
- 年齢/規約/特商法表記
- 資金決済法・景表法・ガチャ規制相当の確認

### 売るなら何を売るか

避ける:

- 最強幻獣の直接販売
- 才能値の直接購入
- 有料限定の強性能

売りやすい:

- 追加派遣枠
- 図鑑/ホームの装飾
- スキン/背景
- 派遣短縮チケット
- 追加倉庫枠
- 捕獲演出スキン
- 支援者バッジ

方針:

**無料でも最後まで遊べる。課金すると気持ちよく便利・見た目が豪華。**

## 8. 実装ロードマップ

### Phase 1: ホーム主役化

目的: ゲームを開いた瞬間に「自分の幻獣がいる」と感じる。

実装:

- Home上部に相棒カード
- 大きな幻獣表示
- 今日のやることカード
- 記章/実績/日課の赤点強化
- メニュー導線を画像アイコン中心に整理

### Phase 2: クイック探索

目的: 移動操作なしで遊べる主導線を作る。

実装:

- 探索先選択
- 探索イベントカード
- 既存Battleへの接続
- 素材/宝箱/会話イベント
- 探索結果画面

### Phase 3: 図鑑カード化

目的: 300体の価値を見える化。

実装:

- 1体詳細カード
- 未発見シルエット
- レア/変異/才能バッジ
- 入手場所
- お気に入り
- シェアカード強化

### Phase 4: 派遣

目的: ログイン理由と放置報酬を作る。

実装:

- 派遣スロット
- 帰還タイマー
- 素材/コイン/レア痕跡
- 派遣帰還演出

### Phase 5: 錬成・素材経済

目的: やりこみの深さを作る。

実装:

- 素材24種
- 器4種
- 継承/才能/進化補助
- 素材周回導線

### Phase 6: 課金準備

目的: 売る前の土台。

実装:

- サーバー保存
- アカウント
- 購入履歴
- 有料通貨と無料通貨の分離
- 規約/表記/法務確認

## 10. 判断基準

今後の実装判断は次の順で決める。

1. 幻獣が魅力的に見えるか
2. 操作数が減るか
3. 毎日戻る理由が増えるか
4. 長期で集めたくなるか
5. 将来の課金導線を壊さないか

この5つに効かない改修は後回しにする。
