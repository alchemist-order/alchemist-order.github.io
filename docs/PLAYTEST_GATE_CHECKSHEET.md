## 4. 判定

- **3問すべてYes** → ✅ゲート通過。GA4設定→ソフトローンチへ
- **1つでもNo** → ❌そのNoの場面を最優先で修正（機能追加は全停止）。修正後に該当区間だけ再プレイ
- 判定に迷う（「Yesだけど弱い」）→ Noとして扱う（初見の他人はあなたより体験が薄くなるため）
