# 品質向上計画（2026-07策定）— 浅さの解消を最優先

成長施策（シェアカード/変異種/週替わり塔）より**先に**実行する。ROADMAP_90D.mdのP0はこの計画で差し替え。
判定基準: 完了後に30分通し遊びで「戦闘で考えた瞬間/ダンジョンで欲をかいた瞬間/技が気持ちいい瞬間」が3つYesなら告知フェーズへ。

## 診断
- 戦闘: 技が数字違いのみ→最適解が「相性良い最大威力連打」。駆け引きの語彙(バフ/先制/ガード)がゼロ
- エフェクト: 全技同一演出。タイプの視覚的アイデンティティ無し
- マップ/街: ゾーニングと高低差が無く「平面」。雰囲気パスは森のみ
- ダンジョン: 選択(寄り道リスク/リワード)と資源圧力が無く「移動」

## パッケージA: 戦闘の深み（最優先）

- A-3 タイプ機械個性: 火=高火力+反動 / 水=ドレイン+防御↑ / 風=先制+連続+速さ↑ / 地=ガード+重一撃 /
  雷=まひ+高会心 / 毒=毒重ね+デバフ / 聖=回復+バフ+状態解除 / 冥=吸収+命中不安高火力 / 錬成=調律(ランクリセット)等

## パッケージD: ダンジョンの攻略化
**実装済(2026-07・Sonnet・実機検証OK)**: 森(forest)に4機能を追加。
- **encounterZones**: `GameMap.zones`(矩形+pool/min/max)。草地A/B=Lv4-6→C=Lv6-9→D=Lv8-11→E=Lv10-13→F(最奥)=Lv11-14+`rarePool`(進化後個体, 15%)。実機で草地F(Lv11-13実測)がベース設定(Lv4-8)を上書きすることを確認。
- **ヌシ幻獣**: 廊下3から分岐する隠し部屋(草地の奥、宝箱1個)を野生マンドラゴ(Lv13,talent6=★★確定)が封鎖。`BattleConfig.forcedSpeciesId/forcedLevel/forcedTalent`+`nushiId`を新設し`makeWild`で固定出現、撃破/捕獲(`onExit`が`caught`もwon扱いに拡張)で`flag=nushi_<id>`が立ち以後は素通り可能(未解決の間はNPCと同様に接触でバトル開始・通過不可)。見た目は`Sprite`1.6倍+CSS常時ループの赤オーラ(`field-zones.css`、index.css非干渉)。実機で撃破→フラグ確認→ヌシがいたマスを通過→部屋の宝箱/スイッチに到達まで確認済み。
- **ルーンスイッチ→ゲート**: ヌシの間の奥にスイッチ、`switch_forest_shortcut`フラグで廊下2⇔廊下4を直結する近道ゲート('X'タイル、実際の通行判定は`GameMap.gates`のflag照合)を解放。実機で解放前後の通行不可/可を確認。
- **消耗設計のシミュ(S3後半)実施済(2026-07・Sonnet、`scripts/sim_dungeon_resource.mjs`)**: 「ボス到達時に資源7割」目標に対し**未達**。最短経路/標準探索/くまなく探索/ヌシ挑戦の全4シナリオでwipe(全滅)率34-86%(目標15%以下を大幅超過)、wipeを除いた到達時HP%も28-59%(目標60-80%を下回る)。
  - **重大な発見①タイプ相性の構造的偏り**: 森は地/毒/風テーマだが、御三家は火/水/風。火のイグニフはwipe率85.3%、水のアクアブ39.7%、風のコグリフ35.3%——**御三家の選択によって最初のダンジョン難易度が大きく変わってしまっている**(火は地に0.5倍・地は火に2倍で構造的に不利)。御三家のどれを選んでも同程度の難易度になるべきという前提が崩れている。
  - **発見②アイテム経済の脆弱性**: 母の贈り物(傷薬×3)だけでは道中の戦闘回数(想定7〜30戦)に対して不足し、切れた後は回復手段がないままジリ貧になりやすい。
  - **発見③パーティ運用でも改善は限定的**: 2体目(相性の良い風タイプ)を仲間に加えても wipe率49.6%と大きくは改善しない。
  - **発見④単純な一方向調整では解決しない**: エンカウント率を18%→12%に下げる実験ではwipe率が悪化(54.9%→70.5%)。エンカ率低下が戦闘機会=経験値機会も減らすため、後半ゾーンにアンダーレベルで突入しやすくなる副作用が「消耗が減る」効果を上回った。宝箱をheal2に強化してもwipe率は改善しなかった(HP%平均は改善)。**単純な数値ノブ1つの調整では70%目標に到達できず、レベリング速度とエンカ率のトレードオフを踏まえた設計変更(ゾーンLv幅の圧縮、御三家ごとの序盤救済、パーティ推奨の明示等)が必要**。
  - 今回はヘッドレスシミュの限界(簡易AI・PP無視・完全な戦略的パーティ運用は非模擬)に達したため**実際の数値変更は行っていない**。実機での通しテストプレイでの検証をFable/Opusに申し送る。
- 担当: F=森v2レイアウト+ヌシ表→O=実装(→実際はSonnetが担当)→S=シミュ/QA(消耗設計の検証は未着手)

# 第2次品質スプリント（2026-07策定・A〜D完了を受けて）

## 診断（感覚ではなく実測から）
パックA〜Dで「仕組みの浅さ」は解消した。残る品質低下要因は、シミュ/QAで実際に観測された次の3つ:
1. **序盤の理不尽**: 火スターターのみwipe率85%(水40/風35)、傷薬3個は道中戦闘数に対し不足、マレアは未シミュのtalent6(シルヴァ初版と同じ構成=詰みの疑い濃厚)
2. **体感テンポと可視性**: 戦闘の待ち時間(O2未実装)。パッケージAで入れた駆け引き(ランク/技タグ/相性)が新規プレイヤーに**見えない**——深みは知覚されて初めて品質になる
3. **署名機構の埋没**: タイトルが「錬金」なのに、錬成は2体揃えてミルカを偶然訪ねるまで存在に気づけない
