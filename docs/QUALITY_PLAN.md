成長施策（シェアカード/変異種/週替わり塔）より**先に**実行する。ROADMAP_90D.mdのP0はこの計画で差し替え。
判定基準: 完了後に30分通し遊びで「戦闘で考えた瞬間/ダンジョンで欲をかいた瞬間/技が気持ちいい瞬間」が3つYesなら告知フェーズへ。

## 診断
- 戦闘: 技が数字違いのみ→最適解が「相性良い最大威力連打」。駆け引きの語彙(バフ/先制/ガード)がゼロ
- エフェクト: 全技同一演出。タイプの視覚的アイデンティティ無し
- マップ/街: ゾーニングと高低差が無く「平面」。雰囲気パスは森のみ
- ダンジョン: 選択(寄り道リスク/リワード)と資源圧力が無く「移動」

- A-3 タイプ機械個性: 火=高火力+反動 / 水=ドレイン+防御↑ / 風=先制+連続+速さ↑ / 地=ガード+重一撃 /
