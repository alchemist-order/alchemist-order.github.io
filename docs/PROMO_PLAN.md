**30分通し遊びゲート合格が全告知の前提条件。** ゲート不合格のまま告知すると「初動で来た人が二度と戻らない」— 初動は一度しか使えない資源。

- [ ] OGP確認 … 済（bg/title.jpg 設定済・実在確認済）

- スクショ・GIFは実プレイ画面のみ。加工は文字乗せまで

### Phase 1: ソフトローンチ（W1）
**目的: 本告知の前に「他人が遊んで詰まる場所」を炙り出す（本人30分ゲートでは拾えない初見の迷いを回収）**
- 個人開発ハッシュタグで控えめに第一報（テンプレ①）。フォロワー外に爆発させない静かな出し方
- 知人・家族2-3人に直接URLを渡して感想回収
- GA4で `starter_chosen`→`gym_defeated` の到達率を確認（離脱点の発見）
- 出た問題をOpusが修正 → 週末までに安定版

### Phase 2: 本告知（W2）
- **火曜または水曜の20-22時**（X個人開発界隈のゴールデン）にメイン告知（テンプレ②、バナー＋GIF付き）
- 同週内にフリゲポータル登録: **ふりーむ！／フリーゲーム夢現／itch.io**（§4）。ポータルは検索流入の資産になる（Xは瞬発、ポータルは持久）
- 土曜に #スクリーンショットサタデー 参加（テンプレ⑤）

### Phase 3: 週次運用（W3〜、F7本体）
**「月曜挑戦・金曜報告」の固定リズム**（習慣化＝フォロー理由を作る）
| 曜日 | 投稿 | テンプレ |
|---|---|---|
| 月曜 朝 | 塔チャレンジ告知「今週は何階まで行ける？」 | ③ |
| 水曜 | 幻獣図鑑紹介（1体ピックアップ、週替わり） | ④ |
| 金曜 夜 | 週のまとめ（更新内容/リプ返し/スコア紹介） | ⑥ |
| 土曜 | #スクリーンショットサタデー | ⑤ |
- リプ・引用でシェアカードを投稿してくれた人には**必ず反応**（初期は1人の共有が10人分の広告）
- S6: Sonnetが図鑑紹介④を4週分まとめてドラフト量産

### Phase 4: アップデート山場（M2〜）
- 週替わり塔（F2→O9）実装時が第二の告知山。「毎週月曜、塔が変わる」でPhase 3のリズムと接続
