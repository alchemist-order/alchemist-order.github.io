策定: 2026-07-17 Fable。ユーザー決定「野生で捕まえられるのは120くらい。あとは錬成・アイテム使用・進化のみ」を全300種に適用する。
[IMPL_O16_STAGE_EXPANSION.md](IMPL_O16_STAGE_EXPANSION.md) を本書で改訂する（O16実装とセットでCodexが実装）。

## 0. なぜこれが効くか

「全部野生で拾える」と錬成・育成が飾りになる。入手経路を分けると:
- 野生120種=**入口は広く**（どのステージでも新顔に会える）
- 進化=**育てる理由**、錬成のみ=**錬成釜を回す理由**、アイテムのみ=**深域に潜る理由**
- 図鑑の残り枠が「どうやって手に入れる?」という**謎**になる（コンプの奥行き）

## 1. 経路マトリクス（全300種・数字は実データで検証済み）

|---|---|---|---|
| 🌿 野生捕獲 | **119** | **stage1全種**（進化なし単独種11を含む） | O16の27ステージ |

- **野生プールは全ステージ tiers:[1] に統一**（O16 §2表の「段階」列は全て1と読み替える）

## 2. 実装（src/game/acquisition.ts 新規 — monsters.jsonは触らない）

```ts
// 入手経路の上書き層。データファイルを変更せず、進化判定の前段でルートを決める。
export type Route = 'wild' | 'levelup' | 'fusion' | 'item'
export function routeOf(id: string): Route {
  const d = species(id)
  if (d.stage === 1) return 'wild'
  if (d.stage === 2) return 'levelup'
  if (!d.id.startsWith('g')) return 'levelup'      // canon stage3 は従来進化
  return hashEven(d.id) ? 'fusion' : 'item'        // g種stage3 を決定的に折半
}
// hashEven: cyrb53(id) & 1 === 0（Math.random禁止・ビルド毎に不変）
```
